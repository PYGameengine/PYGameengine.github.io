系统要求

在安装 PYGameengine 之前，请确保你的系统满足以下要求：

Windows 系统

  • Windows 7 或更高版本（推荐 Windows 10/11）
  • 至少 4GB RAM（推荐 8GB 或更多）
  • 至少 2GB 可用硬盘空间
  • 支持 OpenGL 3.3 或更高版本的显卡
  • Python 3.7 或更高版本

macOS 系统

  • macOS 10.14 (Mojave) 或更高版本
  • 至少 4GB RAM（推荐 8GB 或更多）
  • 至少 2GB 可用硬盘空间
  • 支持 Metal 或 OpenGL 3.3 或更高版本的显卡
  • Python 3.7 或更高版本

Linux 系统

  • Ubuntu 18.04 或其他兼容的 Linux 发行版
  • 至少 4GB RAM（推荐 8GB 或更多）
  • 至少 2GB 可用硬盘空间
  • 支持 OpenGL 3.3 或更高版本的显卡
  • Python 3.7 或更高版本

注意：对于开发 3D 游戏或复杂项目，建议使用更高配置的系统以获得更好的性能。

前置条件

在安装 PYGameengine 之前，你需要先安装 Python 和一些必要的依赖项。

安装 Python

PYGameengine 需要 Python 3.7 或更高版本。你可以从 Python 官方网站 下载并安装最新版本的 Python。

Windows 上安装 Python
  1. 访问 Python 下载页面 并下载最新的 Windows 安装程序。
  2. 运行安装程序，确保勾选 "Add Python to PATH" 选项。
  3. 按照安装向导完成安装。
macOS 上安装 Python

你可以使用 Homebrew 安装 Python：

# 安装 Homebrew（如果尚未安装）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 安装 Python
brew install python
Linux 上安装 Python

大多数 Linux 发行版已经预装了 Python，但你可能需要安装特定版本：

# Ubuntu/Debian
sudo apt update
sudo apt install python3 python3-pip

# Fedora
sudo dnf install python3 python3-pip

# Arch Linux
sudo pacman -S python python-pip

安装必要的系统依赖

Windows

Windows 用户可能需要安装 Visual C++ Redistributable for Visual Studio 2015-2019，可以从 Microsoft 官方网站 下载。

macOS

macOS 用户可能需要安装 Xcode 命令行工具：

xcode-select --install
Linux

Linux 用户可能需要安装一些系统依赖：

# Ubuntu/Debian
sudo apt update
sudo apt install build-essential libgl1-mesa-dev libgles2-mesa-dev

sudo apt install libsdl2-dev libsdl2-image-dev libsdl2-mixer-dev libsdl2-ttf-dev

sudo apt install libopenal-dev libvorbis-dev

安装方法

有多种方法可以安装 PYGameengine，选择最适合你的方法。

方法 1：使用 pip 安装（推荐）

最简单的方法是使用 Python 的包管理器 pip 来安装 PYGameengine：

# 安装最新版本
pip install pygameengine

# 安装特定版本
pip install pygameengine==1.0.0

# 升级到最新版本
pip install --upgrade pygameengine

提示：在某些系统上，你可能需要使用 pip3 而不是 pip。

方法 2：从源代码安装

如果你想使用最新的开发版本或自定义引擎，可以从源代码安装：

# 克隆仓库
git clone https://github.com/pygameengine/pygameengine.git

# 进入目录
cd pygameengine

# 安装依赖
pip install -r requirements.txt

# 安装引擎（开发模式）
pip install -e .

验证安装

安装完成后，你可以通过创建一个简单的脚本来验证 PYGameengine 是否正确安装：

# 创建一个名为 verify_install.py 的文件
from pygameengine import Engine

# 初始化引擎
engine = Engine()

# 打印版本信息
print(f"PYGameengine 版本: {engine.version}")

# 检查引擎组件
components = engine.get_available_components()
print("可用组件:")
for component in components:
    print(f"- {component}")

运行脚本：

python verify_install.py

如果一切正常，你应该会看到引擎版本信息和可用组件列表。

故障排除

如果你在安装过程中遇到问题，可以尝试以下解决方案：

Windows 上的常见问题

  • 问题：找不到 pip 命令
    解决方法：确保在安装 Python 时勾选了 "Add Python to PATH" 选项，或者手动将 Python 和 pip 添加到系统 PATH 中。
  • 问题：安装依赖项时出错
    解决方法：确保安装了 Visual C++ Redistributable，并且使用管理员权限运行命令提示符。

macOS 上的常见问题

  • 问题：OpenGL 相关错误
    解决方法：确保你的 macOS 版本支持 Metal 或 OpenGL 3.3+，并且安装了最新的显卡驱动。
  • 问题：权限问题
    解决方法：尝试使用 sudo 命令或安装到用户目录：pip install --user pygameengine。

Linux 上的常见问题

  • 问题：缺少系统依赖
    解决方法：确保安装了所有必要的系统依赖，如上文所述。
  • 问题：SDL 相关错误
    解决方法：安装 SDL2 开发包：sudo apt install libsdl2-dev（Ubuntu/Debian）。

警告：如果你遇到了无法解决的安装问题，请查看我们的 常见问题 页面或在 GitHub 仓库上提交一个 issue。

卸载方法

如果你需要卸载 PYGameengine，可以使用以下命令：

# 使用 pip 卸载
pip uninstall pygameengine

这将卸载 PYGameengine 包，但不会删除你的项目文件。

提示：如果你想完全移除所有相关文件，你可能还需要手动删除 Python 的 site-packages 目录中的 pygameengine 文件夹，以及任何全局配置文件。
