介绍

PYGameengine 是一款轻量级、高效能的游戏开发引擎，专为 Python 开发者打造。它结合了 Python 语言的简洁性和游戏开发所需的强大功能，让游戏开发变得简单而强大。

无论你是游戏开发新手还是有经验的开发者，PYGameengine 都能为你提供直观的 API 和丰富的工具，帮助你快速构建高质量的 2D 和 3D 游戏。

注意：本指南假设你已经具备基本的 Python 编程知识。如果你是 Python 初学者，建议先学习 Python 基础知识后再开始使用本引擎。

核心特性

PYGameengine 提供了丰富的功能，帮助你创建各种类型的游戏：

  • 跨平台支持：支持 Windows、macOS、Linux 等多种操作系统
  • 2D 和 3D 渲染：强大的渲染系统，支持 2D 精灵和 3D 模型
  • 物理引擎：集成物理系统，支持碰撞检测、重力、弹性等
  • 音频系统：支持音效和背景音乐播放
  • 输入处理：统一的输入系统，支持键盘、鼠标、游戏手柄等
  • 场景管理：灵活的场景和层级管理系统
  • 资源管理：自动加载和管理游戏资源
  • 脚本系统：支持 Python 脚本扩展游戏功能
  • 编辑器：内置可视化编辑器，简化开发流程

快速开始

要开始使用 PYGameengine，你需要先安装引擎，然后创建你的第一个游戏项目。

1. 安装 PYGameengine

你可以通过以下步骤安装 PYGameengine：

# 使用 pip 安装 PYGameengine
pip install pygameengine

详细的安装说明请参阅 安装指南。

2. 创建第一个游戏项目

安装完成后，你可以使用 PYGameengine 的启动器创建一个新的游戏项目：

# 启动 PYGameengine 启动器
python -m pygameengine.launcher

启动器启动后，点击"新建项目"按钮，然后按照向导创建一个新的游戏项目。

3. 项目结构

创建新项目后，你的项目目录结构应该类似于以下内容：

my_game/
├── assets/         # 游戏资源目录
│   ├── models/     # 3D 模型文件
│   ├── textures/   # 纹理文件
│   ├── sounds/     # 音频文件
│   └── fonts/      # 字体文件
├── scenes/         # 场景文件
├── scripts/        # Python 脚本
├── config.py       # 游戏配置
└── main.py         # 游戏入口文件

4. 运行你的第一个游戏

创建项目后，你可以使用以下命令运行你的游戏：

# 在项目目录中运行游戏
cd my_game
python main.py

提示：如果你使用的是 PYGameengine 的内置编辑器，你可以直接点击编辑器中的"运行"按钮来启动游戏。
